# CPP Module 05 – Repetition and Exceptions

## Overview

CPP Module 05 focuses on **exceptions**, **inheritance**, **polymorphism**, and **abstract classes**. Throughout four exercises, a bureaucracy system is built where bureaucrats sign and execute different types of forms while handling errors through exceptions.

## Exercises

* **ex00:** Implemented the `Bureaucrat` class with grade validation and custom exceptions.
* **ex01:** Added the `Form` class and implemented the signing process.
* **ex02:** Introduced the abstract `AForm` class and implemented:

  * `ShrubberyCreationForm`
  * `RobotomyRequestForm`
  * `PresidentialPardonForm`
* **ex03:** Implemented the `Intern` class, capable of creating forms based on their names using a lookup table and function pointers.

## Key Concepts

* Exceptions
* Inheritance
* Abstract classes
* Virtual functions
* Polymorphism
* Dynamic memory management
* Function pointers
* Const correctness
* Object-Oriented Programming (OOP)
